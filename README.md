# Tallao
Talla'o is a web browser which allows users to check into their laundry orders. It allows to communicate, inform about the order.

Version of Frameworks and APIs used

PHP - 7.2.27
Jquery - 3.5.1
RequireJS - 2.3.6
Font Awesome 4.7.0
Bootstrap CSS - 4.4.1
Bootstrap JS - 4.3.1
Popper JS - 1.14.7
